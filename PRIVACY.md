# Privacy Policy - Octara

**Last updated:** March 10, 2026

## Overview

Octara is a Chrome extension that provides text translation, pronunciation, and AI sentence analysis. This privacy policy explains how Octara handles user data.

## Data Collection

Octara does **not** collect, store, or transmit any personal information. Specifically, we do not collect:

- Personal identification information (name, email, address)
- Browsing history or web activity
- Location data
- Financial or payment information
- Health or authentication data

## Data Storage

The following data is stored **locally in your browser only** using Chrome's storage API:

- API keys you enter (Gemini, DeepL)
- Your preference settings (translation engine, target language, TTS voice/speed)
- Daily API usage statistics

This data never leaves your browser and is not accessible to us or any third party.

## External Communications

Octara communicates with the following external services **only** to process translation requests:

| Service | Purpose | Data Sent |
|---|---|---|
| Google Gemini API | AI translation & sentence analysis | Selected text only |
| DeepL API | Translation | Selected text only |
| MyMemory API | Translation (fallback) | Selected text only |

No personal information, browsing data, or metadata is included in these requests.

## Third-Party Data Sharing

We do not sell, trade, or transfer any user data to third parties.

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page.

## Contact
