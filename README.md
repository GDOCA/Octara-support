# Octara-support
Octara - Smart Translation Extension

  Instantly translate text by simply dragging to select it on any web page.

  Key Features:
  - Instant drag-to-translate: Select text and get translation results in a popup instantly
  - Multi-language support: Translate between English, Japanese, Chinese, Spanish, French, and more
  - Text-to-Speech (TTS): Listen to the pronunciation of translated text
  - AI Sentence Analysis: Get detailed sentence breakdowns and explanations powered by Google Gemini AI
  - DeepL Translation: Option to use the high-quality DeepL translation engine

  How to Use:
  1. Click the extension icon to activate translation mode.
  2. Drag to select any text on a web page.
  3. View translations, listen to pronunciation, and explore AI analysis in the popup.

  Settings:
  - Register your Google Gemini API key and DeepL API key in the options page.
  - Customize your target language and preferred translation engine.

  All data is stored locally in your browser only. No personal information is collected or transmitted externally.

  주요 기능:
  - 드래그만으로 즉시 번역: 텍스트를 선택하면 팝업으로 번역 결과를 바로 확인
  - 다국어 지원: 영어, 일본어, 중국어, 스페인어, 프랑스어 등 다양한 언어 번역
  - 발음 듣기(TTS): 번역된 텍스트의 발음을 바로 들을 수 있음
  - AI 문장 분석: Google Gemini AI를 활용한 문장 구조 분석 및 상세 해설
  - DeepL 번역 지원: 고품질 DeepL 번역 엔진 선택 가능

  사용 방법:
  1. 확장 프로그램 아이콘을 클릭하여 번역 기능을 활성화합니다.
  2. 웹 페이지에서 번역하고 싶은 텍스트를 드래그합니다.
  3. 팝업에서 번역 결과, 발음 듣기, AI 분석을 이용합니다.

  설정:
  - 옵션 페이지에서 Google Gemini API 키와 DeepL API 키를 등록하여 사용합니다.
  - 번역 대상 언어와 번역 엔진을 자유롭게 설정할 수 있습니다.
