# Octara - 스마트 번역 확장 프로그램

Octara는 웹 페이지에서 텍스트를 드래그하면 바로 번역 결과를 확인할 수 있는 Chrome 확장 프로그램입니다.

---

## 설치 방법

1. [Chrome 웹 스토어](#)에서 Octara를 설치합니다. (링크 준비 중)
2. 설치 후 Chrome 툴바에 Octara 아이콘이 나타납니다.
3. 아이콘이 보이지 않으면 툴바의 퍼즐 아이콘(확장 프로그램)을 클릭하고 Octara를 고정하세요.

---

## 시작하기

### 1. API 키 설정

Octara를 사용하려면 최소 하나의 API 키를 등록해야 합니다.

**설정 페이지 열기:**
- Octara 아이콘 우클릭 → **옵션**
- 또는 `chrome://extensions` → Octara 찾기 → **세부정보** → **확장 프로그램 옵션**

#### Gemini API 키 (AI 기능 사용 시 필수)

1. [Google AI Studio](https://aistudio.google.com/apikey)에 접속합니다.
2. Google 계정으로 로그인합니다.
3. **"Create API Key"** 버튼을 클릭합니다.
4. 생성된 API 키를 복사하여 Gemini API 키 입력란에 붙여넣습니다.
5. **저장**을 클릭하면 키가 자동으로 검증됩니다.

> 무료 티어: 분당 15회, 일 1,500회 요청 가능

#### DeepL API 키 (선택사항, 고품질 번역용)

1. [DeepL API](https://www.deepl.com/en/pro#developer) 페이지에 접속합니다.
2. **"Sign up for free"**를 클릭하여 DeepL API Free 플랜으로 가입합니다.
3. 본인 인증을 위해 신용카드/체크카드를 등록합니다 (과금되지 않습니다).
4. [Account > API Keys](https://www.deepl.com/your-account/keys)에서 키를 복사합니다.
5. DeepL API 키 입력란에 붙여넣고 **저장**을 클릭합니다.

> 무료 티어: 월 500,000자, 자동 유료 전환 없음

**참고:** API 키는 브라우저 로컬에만 저장됩니다. 30일 후 만료되며 재입력이 필요합니다.

---

### 2. 번역 활성화

툴바의 Octara 아이콘을 클릭하여 번역 ON/OFF를 전환합니다.

- **ON**: 배지에 `ON`(파란색) 표시 — 텍스트를 드래그하면 번역됩니다.
- **OFF**: 배지 사라짐 — 확장 프로그램이 비활성 상태입니다.

## 기능

### 드래그 번역

1. 번역이 **ON** 상태인지 확인합니다 (아이콘 클릭).
2. 웹 페이지에서 영어 텍스트를 드래그하여 선택합니다.
3. 팝업이 나타나며 한국어 번역 결과를 보여줍니다.

> 기본적으로 영어가 포함된 텍스트만 번역됩니다.

### 발음 듣기 (TTS)

번역 팝업에서 🔊 버튼을 클릭하면 원문의 발음을 들을 수 있습니다.

옵션 페이지에서 TTS 음성과 속도를 변경할 수 있습니다:

1. **옵션** → **TTS 음성** 섹션으로 이동합니다.
2. 드롭다운에서 원하는 음성을 선택합니다:
   - Chrome OS US English 1~7
   - Google UK English Female / Male
3. **속도** 슬라이더로 재생 속도를 조절합니다 (0.5배 ~ 2.0배, 기본값 0.9배).
4. **테스트** 버튼을 클릭하여 선택한 음성과 속도를 미리 들어볼 수 있습니다.

선택한 설정은 자동으로 저장되며, 이후 모든 발음 듣기에 적용됩니다.

### AI 문장 분석

1. 번역 후 팝업에서 **"AI 해석"** 버튼을 클릭합니다.
2. 다음 항목이 포함된 상세 분석 패널이 열립니다:
   - **문장 구조** — 주어, 동사, 목적어, 수식어 분석
   - **핵심 어휘** — 주요 단어와 숙어의 뜻과 용법
   - **문법 포인트** — 시제, 구문, 절 등 문법 설명
   - **번역 해설** — 직역과 의역의 차이 설명
   - **유사 표현** — 같은 의미의 다른 영어 표현 2~3개 제시

> Gemini API 키가 필요합니다.

### 커스텀 번역 (다국어)

단축키를 사용하여 선택한 텍스트를 원하는 언어로 번역할 수 있습니다:

| OS | 단축키 |
|---|---|
| macOS | `Ctrl + S` |
| Windows/Linux | `Alt + S` |

영어뿐만 아니라 모든 언어의 텍스트를 컨텍스트 메뉴에서 설정한 대상 언어로 번역합니다.

## 설정 (컨텍스트 메뉴)

Octara 아이콘을 우클릭하면 추가 설정에 접근할 수 있습니다:

### 번역 엔진 선택

사용할 번역 엔진을 선택합니다:

| 엔진 | 설명 |
|---|---|
| **자동** (기본값) | DeepL → Gemini → MyMemory 순서로 시도 |
| **DeepL** | 고품질 신경망 번역 (API 키 필요) |
| **Gemini** | Google AI 기반 번역 (API 키 필요) |
| **MyMemory** | 무료 번역, API 키 불필요 |

### 번역 대상 언어

커스텀 번역(`Ctrl+S` / `Alt+S`)의 대상 언어를 설정합니다:

- English, 한국어, 日本語, 中文(简体)
- Español, Français, Deutsch, Русский, Português

---

## 사용량 통계

설정 페이지에서 두 API의 일일 사용량을 확인할 수 있습니다:

**Gemini:**
- 요청 수 (일일 한도: 1,500회)
- 토큰 사용량 (프롬프트 / 응답 / 총합)
- 프로그레스 바 (70%에서 노란색, 90%에서 빨간색 경고)

**DeepL:**
- 요청 수 및 글자 사용량 (일일)
- 월간 글자 사용량 대비 한도 (500,000자)
- 프로그레스 바 경고 표시

통계는 매일 초기화됩니다. DeepL 월간 사용량은 DeepL API에서 직접 가져옵니다.

## 문제 해결

| 문제 | 해결 방법 |
|---|---|
| 번역 팝업이 나타나지 않음 | 번역이 ON 상태인지 확인하세요 (아이콘 클릭). 영어 텍스트를 선택했는지 확인하세요. |
| "API 키가 설정되지 않았습니다" | 옵션 페이지에서 Gemini API 키를 등록하세요. |
| "API 키가 만료되었습니다" | API 키는 30일 후 만료됩니다. 옵션에서 키를 재입력하세요. |
| "확장 프로그램이 업데이트되었습니다" | 페이지를 새로고침하세요 (F5). |
| DeepL 번역이 작동하지 않음 | DeepL API 키가 등록되어 있고 유효한지 확인하세요. |
| 특정 사이트에서 번역이 안 됨 | 일부 사이트는 콘텐츠 스크립트를 차단할 수 있습니다. 페이지를 새로고침해보세요. |

## 개인정보 보호

- 모든 API 키와 설정은 **사용자의 브라우저 로컬에만 저장**됩니다.
- 개인정보를 수집하거나 외부로 전송하지 않습니다.
- 확장 프로그램이 통신하는 외부 서비스:
  - Google Gemini API (번역 및 AI 분석)
  - DeepL API (번역)
  - MyMemory API (번역)
